# Project Overview

## What we're building
A document management tool that lets small teams create, organise, and search shared notes. Think Notion but simpler, with better keyboard support.

## Target users
Small engineering and product teams (5–20 people) who find Notion too heavy and Google Docs too unstructured.

## Core features (v1)
1. Create and edit markdown documents
2. Organise into workspaces and folders
3. Full-text search across all documents
4. Share with a link (view-only or edit)

## Tech stack
- Frontend: React + Vite
- Backend: FastAPI (Python)
- Storage: PostgreSQL + S3 for attachments
- Auth: OAuth 2.0 (Google + GitHub)

## Status
Alpha — internal team only. Launch target: July 2026.
